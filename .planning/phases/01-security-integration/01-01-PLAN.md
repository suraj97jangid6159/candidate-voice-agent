<objective>
Implement the three-tiered security architecture for the AI voice representative agent:
1. Tier 1 (Scope Guards): Whitelist validation ensuring call transfers target only the candidate's verified number in the database, blocking premium numbers.
2. Tier 2 (Guardrail Filters): Scanners for prompt injection (input transcript checks) and unauthorized agreement commitments (output response checks).
3. Tier 3 (Defensive Prompts): System instructions containing clear boundary rules.
4. Security Auditing: Database table `security_logs` logging all security violations and broadcasting them.
</objective>
